OctoStore releases

Version notes for product positioning, docs, and API safety features.

v0.10.0

Coordination-first positioning

This release reframes OctoStore around the real primitive: self-hostable distributed locks and leases over HTTP for local and multi-machine coordination.

  • homepage updated to lead with coordination, not just agents
  • docs index reorganized so the conceptual guides come before the API reference
  • new narrative docs added for overview, coordination patterns, local coordination, multi-machine coordination, agent coordination, and examples
  • README rewritten to match the broader product story
v0.9.x

Safety and durability groundwork

  • namespace-scoped lock access enforcement
  • WAL-backed lock recovery and health storage telemetry
  • remaining namespace test cleanup
