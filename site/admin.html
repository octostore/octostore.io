Set the endpoint env var to enable OTEL export to your collector:

OTEL_EXPORTER_OTLP_ENDPOINT=http://your-collector:4317
OTEL_SERVICE_NAME=octostore          # optional, defaults to "octostore"
OTEL_SDK_DISABLED=false              # optional

Works with any OTEL-compatible backend:
Grafana Tempo · Jaeger · Honeycomb · OTEL Collector
